## Design Context

### Users
IT support staff and employees at an educational institution (коллеж). Agents handle technical issues, admins manage the system, users submit tickets for IT problems.

### Brand Personality
Efficient, professional, trustworthy. A reliable IT service desk that gets things done without fuss.

### Aesthetic Direction
- **Dark theme reference**: Linear.app, YouTube Studio
- **Background**: #0A0A0A (body), #1A1A1A (cards/surfaces)
- **Borders**: rgba(255,255,255,0.08)
- **Single accent**: #3B82F6 (blue)
- **Typography**: Playfair Display for headings/brand, Geist Sans for UI
- **Light theme**: Clean white surfaces, #D1D5DB input borders
- **No**: purple accents, gradient text, glassmorphism, hero metrics

### Design Principles
1. **Information-first** — UI serves data, not decoration
2. **Consistent tokens** — All colors from CSS custom properties
3. **Subtle dark theme** — Low-contrast borders, muted text, one accent color
4. **Professional typography** — Clear hierarchy with display + sans pairing
